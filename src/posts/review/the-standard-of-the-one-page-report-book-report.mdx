# 👩🏻‍💻 책 내용 간단 정리
<br />

## 보고서가 중요한 이유
보고는 회사의 언어, 다른 업무의 기반, 인간성 평가로도 이어지기에 중요하다
<br />

## 보고의 목적과 결론 한 문장 요약
<ul>
    <li>상황 보고 : 어떤 상황인 지 상상의 이해를 돕는 명확한 정보 전달 = 상대방 질문에 대한 ‘답’ 한 문장</li>
    <li>요청 보고 : 상대방이 읽고 바로 실행하도록 명확한 액션 플랜 전달 = ‘요청’ 한 문장</li>
    <li>제안 보고 : 상대방 납득을 위한 문제 공감, 실행을 위한 명확한 액션 플랜 = ‘너의 (문제)를 위한 나의 (제안)’ 한 문장</li>
</ul>
<br /> <br />

## 핵심 요약 훈련
<table>
    <tr>
        <th scope="col">날짜</th>
        <th scope="col">이름</th>
    </tr>
    <tr>
        <td>What First</td>
        <td>1. [결론] 결론이 뭐야? <br /> 2.[근거 / 사례] 근거는 뭐야? <br />  3. [진행 / 요청 / 대책] 어쩌라고? <br /> = 무조건 결론부터</td>
    </tr>
    <tr>
        <td>2What</td>
        <td>1. [결론 / 현상] 뭐? <br /> 2. [요청 / 제안 / 적용점] 그래서 뭐? <br /> = 요점 / 요청이 있어야 이해가 되고 진행이 된다</td>
    </tr>
    <tr>
        <td>3W</td>
        <td>1. [문제] Why? <br /> 2. [원인] Why so? <br /> 3. [제안] What <br /> = 위 3가지를 계속해서 반복</td>
    </tr>
</table>

## 보고 문장 Tip 10가지
<ul>
    <li>개조식 : 앞에 번호 붙이기.</li>
    <li>범주화 : 동일한 항목끼리 묶기.</li>
    <li>쪼개기 : 추상적인 것들을 프레임을 가지고 날카롭게 쪼개기.</li>
    <li>제목 : 제목은 명분 있는 목표 / 얻을 수 있는 이익을 담기.</li>
    <li>명사형 정리 : ~함, 조사보단 명사형으로 마무리.</li>
    <li>객관적 근거 : 객관적 자료 제시</li>
    <li>숫자와 그래프 : 숫자와 그래프로로 수치화하기.</li>
    <li>구어체 vs 문어체 : 구어체 {'< 문어체'} (단, 필요 이상의 어려운 문어체는 쉽게 수정하기)</li>
    <li>축약어 : 눈치껏 상황에 맞춰 축약어 사용하기</li>
    <li>2가지 보고 센스 : 중간 보고 (전체 가안 및 방향성 검토) 를 거치는 2차 보고</li>
</ul>
